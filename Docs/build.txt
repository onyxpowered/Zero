Build

Zero Build compiles and bundles any project type.
it uses Zero's own build engines and framework detection,
all written and owned by Zero. Zero makes every decision.


supported languages

  Node.js / TypeScript    detected via package.json
  Python                  detected via requirements.txt, pyproject.toml, setup.py
  Go                      detected via go.mod
  Ruby                    detected via Gemfile
  Rust                    detected via Cargo.toml
  Static sites            detected via index.html (no backend files)


detection

  Zero auto-detects your project type. it checks for language markers
  first, then runs Zero's framework detection to identify the specific
  framework in use (Next.js, React, Vue, Django, Rails, etc.).

  you never need to configure the engine. Zero picks the right one.


usage

  zero build              builds your project to dist/
  zero build --out=path   builds to a custom output directory


dev mode

  zero dev

  runs your project locally with hot-reloading where applicable.
  each language uses its native tooling:

    Node.js    tsx --watch
    Python     python (main.py / app.py / server.py)
    Go         go run .
    Ruby       ruby app.rb / bundle exec rackup
    Rust       cargo run
    Static     built-in HTTP server on :3000


deploy

  zero deploy

  builds your project and ships it.
  deployment targets are configured in zero.config.ts.


custom engines

  any code can register a build engine at runtime:

    import { register } from '@zero/engine/build'

    register({
      name:   'my-engine',
      detect: root => existsSync(join(root, 'my.lock')),
      build:  async (root, info, opts) => { ... },
      dev:    async (root, info) => { ... },
    })

  registered engines take priority over the built-in set.


zero.config.ts

  every app has a zero.config.ts at its root.
  configure build options, deployment targets, and resource thresholds:

    import type { ZeroConfig } from '@zero/engine'

    const config: Partial<ZeroConfig> = {
      plugPort: 7770,
      throttle: {
        ram:  { warning: 70, critical: 85 },
        cpu:  { warning: 60, critical: 80 },
      },
      deploy: {
        target: 'zero',
      },
    }

    export default config
